# Meeting Scheduler

## How to run it

There are two ways.
1. Build the project and run the package `meeting-scheduler-0.1-jar-with-dependencies.jar` with input file path as parameter.
2. Use maven and run `exec:java -Dexec.mainClass=dlu.meetingscheduler.Main -Dexec.args=src/main/resources/meetings` at the right directory.
